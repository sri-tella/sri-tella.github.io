About the Project

Testing website with screen magnifier seemed like an easy task, but it was not from a normal user perspective. It was quite challenging at first to understand the various adjustments such as maximum zoom and touchpad sensitivity. I felt that picture in picture view for magnifier was better compared to the full screen and split screen view adjustments on Mac. Some of the challenges I have faced were obscurity of the content, clicking an element on the page with the magnifier on and pixelation and distortion of the text when we zoom closer to maximum.

For the low vision mode, I have increased font size for better readability, made the size of the elements larger, ensured to implement a high contrast mode for better access of elements on the page and Increased the spacing between the paragraphs for better overview of the content
