  • Instructions for using the app

    • I have implemented a 5 key, 3 key and single key interfaces to operate the calculator.

    For the 5 key interface, the following keys resemble the actions

    • a - select previous
    • d - select next
    • s - click selected item
    • w - select up
    • x - select down

    For the 3 key interface, the following keys resemble the actions

    • j - select previous
    • l - select next
    • k - click selected item

    For the single key interface, the g key resembles the following actions

    • g - select next
    • long press on g key - clicks on a selected item

  • Test Protocol

    • My test protocol includes comparing the three built interfaces with each other by conducting three trials for each interface and testing condition, using fingers, stylus held in the mouth and the thumb or toe of the foot

    The equations being tested are as follows:

    • 152 + 87 - 46 = 193
    • 10 * 4 / 2 = 20
    • 20 / 5 = 9 - 4 = 9
    • 100 * 3 - 47 = 253
    • 208 / 8 + 72 = 98

  • Results

    • For the 5 key interface the results were as follows:

    • Using fingers - For three trials, it took an average time of 2 minutes 16 seconds and 7 total errors
    • Using mouth and stylus - For three trials, it took an average time of 4 minutes 18 seconds and 11 total errors
    • Using toe - For three trials, it took an average time of 5 minutes 39 seconds and 16 total errors

    For the 3 key interface the results were as follows:

    • Using fingers - For three trials, it took an average time of 2 minutes 26 seconds and 4 total errors
    • Using mouth and stylus - For three trials, it took an average time of 6 minutes 10 seconds and 13 total errors
    • Using toe - For three trials, it took an average time of 4 minutes 09 seconds and 9 total errors

    For the single key interface the results were as follows:

    • Using fingers - For three trials, it took an average time of 4 minutes 14 seconds and 13 total errors
    • Using mouth and stylus - For three trials, it took an average time of 6 minutes 37 seconds and 20 total errors
    • Using toe - For three trials, it took an average time of 5 minutes 05 seconds and 15 total errors

  • Conclusion

    • I felt that operating a 5 key interface was much easier during the three testing conditions considering the average time taken. As the number of keys in the interface are reduced, the difficulty in operating them increases. Considering the number of errors, 5 key interface has lesser compared to 3 key interface. And single key interface took more time and has more errors compared to the other two interfaces.

    Comparing the different testing conditions, using fingers resulted in lesser average time and lesser errors. Mouth and stylus had a longer average time and high number of errors. And toe input method had higher average time and errors compared to using fingers as input.
